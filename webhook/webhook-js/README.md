# webhook-js

JavaScript reimplementation of [webhook](../webhook/)

## Building
```sh
npm install
npm run build # produces dist/webhook-js.wasm
```

## Deplying and running with Obelisk
```sh
# in repo root
obelisk server run --config obelisk-local-js-webhook.toml
```
